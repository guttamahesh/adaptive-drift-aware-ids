# Adaptive Concept Drift-Aware Intrusion Detection System for IoT Networks

## Objective
Develop an adaptive intrusion detection system capable of handling evolving IoT traffic patterns and concept drift using machine learning techniques.

## Possible Datasets
- CICIoT2023
- UNSW-NB15
- Bot-IoT
- TON_IoT

## Planned Workflow
1. Dataset preprocessing
2. Baseline ML model development
3. Concept drift simulation and detection
4. Adaptive retraining mechanism
5. Performance evaluation and comparison

- Python
- Scikit-learn
- Pandas
- Jupyter Notebook
- Git & GitHub
